# Zomato Clone

## Project Description

Zomato Clone is a responsive food delivery landing page inspired by the Zomato website. The project is built using HTML5 and CSS3 and showcases a modern user interface with a hero section, navigation bar, search functionality, and service cards.

## Features

- Responsive design for desktop and mobile devices
- Attractive hero section with background image
- Navigation menu with login and signup options
- Restaurant and food search bar
- Service cards for Order Online, Dining, and Nightlife
- Clean and modern UI design

## Technologies Used

- HTML5
- CSS3
- Google Fonts (Poppins)
- Unsplash Images

## Purpose

This project was developed to practice front-end web development skills, including page layout design, responsive styling, flexbox, and modern UI development.

## Future Enhancements

- Add JavaScript for interactive search functionality
- Integrate restaurant data using APIs
- Add user authentication pages
- Connect with a backend database
- Implement online food ordering features
